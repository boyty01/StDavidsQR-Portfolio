Parc Dewi Sant Navigation System

QR-enabled indoor navigation and wayfinding solution

JavaScript HTML/CSS QR Code Generation Google Maps API C# ASP.NET backend SQLite

Project Overview

A comprehensive navigation system designed for Parc Dewi Sant, a multi-tenant business complex. The solution addresses the common challenge visitors face when attending meetings or events in large buildings with multiple companies and room locations.

The system bridges outdoor and indoor navigation, guiding visitors from their starting point to the correct car park, and then providing step-by-step indoor wayfinding to their final destination within the building.

Key Features

Visitor Experience

  • Event/Company Selection - Landing page with dropdown menu of all available destinations within the building
  • QR Code Distribution - Generate and display unique QR codes for each location, enabling easy sharing and mobile access
  • Dual Navigation Options - Choose between Google Maps integration for driving directions or direct web-based navigation
  • Car Park Guidance - Automatically directs visitors to the appropriate car park for their destination
  • Indoor Wayfinding - Step-by-step visual guide with photos showing the route from car park to specific room
  • Mobile-First Design - Optimized for on-the-go access via smartphone

Admin Backend

  • Content Management - Add, edit, and remove navigation routes and destinations
  • Image Upload - Upload and manage wayfinding photos for each route step
  • QR Code Generation - Automatically generate QR codes for any navigation route
  • Real-Time Updates - Changes reflect immediately on the public-facing site

Technical Implementation

Built with a focus on simplicity and reliability, the system uses vanilla JavaScript for the frontend with a lightweight backend for content management. QR codes are dynamically generated and linked to specific navigation routes, while the Google Maps API integration provides seamless outdoor navigation.

The admin interface allows non-technical staff to maintain navigation data without developer intervention, reducing ongoing maintenance overhead.

Problem Solved

Parc Dewi Sant hosts multiple companies and events, making it difficult for first-time visitors to navigate efficiently. Traditional printed directions become outdated quickly and don't account for different entry points or parking areas.

This system provides a scalable, always-updated solution that can be shared via QR codes on invitations, emails, or physical signage, significantly improving the visitor experience and reducing reception desk inquiries.
